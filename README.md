# Exercism Solutions

This repository contains my solutions to various exercises from [Exercism](https://exercism.org), a platform for learning and practicing programming through hands-on coding challenges.

Each folder corresponds to a language track (e.g. `javascript`, `python`, `typescript`) and contains individual exercises with their test files and documentation.

## Quick Workflow

Use the repo-level `./ex` command to keep the workflow simple:

```bash
./ex new hello-world
./ex test hello-world
./ex submit hello-world
```

It assumes the `javascript` track by default, installs dependencies automatically when needed, and also works from inside an exercise directory:

```bash
cd javascript/hello-world
../../ex test
../../ex submit
```

If you want another track, use `track/exercise` explicitly:

```bash
./ex test typescript/hello-world
```

## 🧠 Purpose

I use Exercism to:

* Practice writing clean, idiomatic code.
* Explore language-specific best practices.
* Apply test-driven development (TDD).
* Track my growth as a developer over time.
